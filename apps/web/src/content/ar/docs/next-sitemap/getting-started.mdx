## ما هو @onruntime/next-sitemap؟

`@onruntime/next-sitemap` هو مولد خرائط مواقع قوي مصمم بشكل خاص لتطبيقات Next.js. يقوم تلقائيًا بإنشاء ملفات `sitemap.xml` و `robots.txt` لموقع Next.js الخاص بك.

<Callout type="info" title="المميزات الرئيسية">
  - إنشاء خرائط مواقع تلقائي للـ App Router
  - دعم للطرق الديناميكية
  - أولوية وقابلية تغيير قابلة للتخصيص
  - إنشاء robots.txt
  - دعم لخرائط المواقع متعددة اللغات
</Callout>

## لماذا تستخدمه؟

### تحسين SEO

تساعد خرائط المواقع محركات البحث على اكتشاف وفهرسة صفحاتك بشكل أكثر كفاءة:

- **فهرسة أسرع** - تستطيع محركات البحث العثور على جميع صفحاتك
- **زحف أفضل** - أبلغ الزاحفين عن تحديثات الصفحات
- **تلميحات الأولوية** - قم بتوجيه الزاحفين إلى الصفحات المهمة

### تكامل Next.js

مبني خصيصًا لـ Next.js، إنه يفهم:

- هيكل App Router
- الطرق الديناميكية
- التوليد الثابت
- التوجيه الدولي

## مثال سريع

```tsx
// app/sitemap.ts
import { generateSitemap } from "@onruntime/next-sitemap/app";

export default async function sitemap() {
  return generateSitemap({
    baseUrl: "https://example.com",
  });
}
```

هذا ينشئ خريطة موقع كاملة تشمل جميع صفحاتك الثابتة والديناميكية.

## الناتج الناتج

يولد الحزمة خرائط مواقع XML القياسية:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://example.com/</loc>
    <lastmod>2024-01-15</lastmod>
    <changefreq>يومي</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://example.com/about</loc>
    <lastmod>2024-01-10</lastmod>
    <changefreq>شهري</changefreq>
    <priority>0.8</priority>
  </url>
</urlset>
```
