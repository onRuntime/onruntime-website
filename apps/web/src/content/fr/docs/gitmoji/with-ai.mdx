Installez notre plugin gitmoji pour [Claude Code](https://claude.ai/code) pour generer des commits suivant la convention.

## Installer le Plugin

Ajoutez le marketplace onRuntime et installez le plugin :

```bash
# Ajouter le marketplace onRuntime
/plugin marketplace add https://onruntime.com/plugins

# Installer le plugin gitmoji
/plugin install gitmoji@onruntime
```

Ou installez directement depuis le menu :

1. Lancez `/plugin` dans Claude Code
2. Allez dans l'onglet **Discover**
3. Ajoutez le marketplace : `https://onruntime.com/plugins`
4. Cherchez "gitmoji" et installez

## Utilisation

Lancez la commande `/commit` pour creer un commit :

```
/commit
```

```
/commit corriger le bug de login
```

Claude Code analysera vos changements et creera un commit suivant la convention gitmoji.

<Callout type="info" title="Commande Plugin">
  La commande `/commit` recupere automatiquement votre git status et diff pour generer le message de commit approprie.
</Callout>

## Alternative : CLAUDE.md

Si vous preferez ne pas installer le plugin, ajoutez ceci au `CLAUDE.md` de votre projet :

```markdown
## Convention de Commit

Utiliser les gitmoji conventional commits :

\`\`\`
<gitmoji> <type> <description>
\`\`\`

Types : add, fix, improve, update, remove, refactor, rename, move, upgrade, downgrade

Regles :
- Ecrire en minuscules
- Utiliser le mode imperatif ("add" pas "added")
- Un seul changement logique par commit
```
