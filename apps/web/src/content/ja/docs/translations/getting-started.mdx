## @onruntime/translationsとは？

`@onruntime/translations`は、ReactおよびNext.jsアプリケーション向けに設計された軽量で型安全な国際化（i18n）ライブラリです。フルTypeScriptサポートを持つ翻訳管理用のシンプルなAPIを提供します。

<Callout type="info" title="主な機能">
  - TypeScriptによる型安全な翻訳
  - ReactとNext.js（App Router）に対応
  - 名前空間ベースの整理
  - 変数補完
  - 軽量でパフォーマンスが高い
</Callout>

## なんで使うの？

### 型安全

無効な翻訳キーを使用したときにコンパイル時エラーが発生します：

```typescript
const { t } = useTranslation("common");

t("greeting");        // ✅ 有効なキー
t("invalid_key");     // ❌ TypeScriptエラー
```

### シンプルなAPI

翻訳に必要なフックがたった2つです：

```typescript
// クライアントコンポーネント
const { t } = useTranslation("namespace");

// サーバーコンポーネント（Next.js）
const { t } = await getTranslation("namespace");
```

### 整理された翻訳

名前空間を使って翻訳を整理できます：

<Files>
  <Folder name="locales" defaultOpen>
    <Folder name="en" defaultOpen>
      <File name="common.json" />
      <File name="auth.json" />
      <File name="dashboard.json" />
    </Folder>
    <Folder name="fr">
      <File name="common.json" />
      <File name="auth.json" />
      <File name="dashboard.json" />
    </Folder>
  </Folder>
</Files>

## 簡単な例

```tsx
// locales/en/common.json
{
  "greeting": "こんにちは、{name}!"
}

// コンポーネント
import { useTranslation } from "@onruntime/translations/react";

function Welcome({ name }) {
  const { t } = useTranslation("common");

  return <h1>{t("greeting", { name })}</h1>;
}
```
