## Basisgebruik

### De useTranslation Hook

De `useTranslation` hook is jouw primaire manier om vertalingen in React-componenten te behalen:

```tsx
import { useTranslation } from "@onruntime/translations/react";

function MyComponent() {
  const { t, locale } = useTranslation("common");

  return (
    <div>
      <p>Huidige locale: {locale}</p>
      <h1>{t("title")}</h1>
    </div>
  );
}
```

### Variabele Interpolatie

Geef variabelen door aan je vertalingen:

```json
// locales/en/common.json
{
  "greeting": "Hallo, {name}!",
  "items_count": "Je hebt {count} items"
}
```

```tsx
function Greeting({ name, itemCount }) {
  const { t } = useTranslation("common");

  return (
    <>
      <p>{t("greeting", { name })}</p>
      <p>{t("items_count", { count: itemCount })}</p>
    </>
  );
}
```

## Meerdere Namespaces

Je kunt meerdere namespaces gebruiken in dezelfde component:

```tsx
function Dashboard() {
  const { t: tCommon } = useTranslation("common");
  const { t: tDashboard } = useTranslation("dashboard");

  return (
    <div>
      <h1>{tDashboard("title")}</h1>
      <button>{tCommon("buttons.save")}</button>
    </div>
  );
}
```

## Taal Veranderen

Gebruik de `setLocale` functie om de huidige locale te veranderen:

```tsx
import { useTranslation } from "@onruntime/translations/react";

function LanguageSwitcher() {
  const { locale, setLocale } = useTranslation("common");

  return (
    <select value={locale} onChange={(e) => setLocale(e.target.value)}>
      <option value="en">Engels</option>
      <option value="fr">Français</option>
      <option value="de">Deutsch</option>
    </select>
  );
}
```

## Geneste Sleutels

Toegang tot geneste vertalingssleutels met puntnotatie:

```json
// locales/en/common.json
{
  "nav": {
    "home": "Home",
    "about": "Over",
    "contact": "Contact"
  }
}
```

```tsx
function Navigation() {
  const { t } = useTranslation("common");

  return (
    <nav>
      <a href="/">{t("nav.home")}</a>
      <a href="/about">{t("nav.about")}</a>
      <a href="/contact">{t("nav.contact")}</a>
    </nav>
  );
}
```

### Houd Sleutels Consistent

Gebruik consistente naamgevingsconventies in alle locales:

```json
// ✅ Goed - consistente structuur
{
  "page_title": "Dashboard",
  "actions": {
    "save": "Opslaan",
    "cancel": "Annuleren"
  }
}

// ❌ Vermijd - inconsistent
{
  "pageTitle": "Dashboard",
  "save_button": "Opslaan",
  "cancelBtn": "Annuleren"
}
```
