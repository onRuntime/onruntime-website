## TranslationsProvider

مكون المزود الذي يلتف حول تطبيقك.

```tsx
import { TranslationsProvider } from "@onruntime/translations/react";

<TranslationsProvider
  locale="ar"
  locales={{ en, fr }}
  fallbackLocale="en"
>
  {children}
</TranslationsProvider>
```

### Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `locale` | `string` | نعم | اللغة النشطة الحالية |
| `locales` | `Record<string, object>` | نعم | كائن يحتوي على جميع ترجمات اللغة |
| `fallbackLocale` | `string` | لا | اللغة الاحتياطية عند غياب المفتاح |
| `children` | `ReactNode` | نعم | مكونات الأطفال |

## useTranslation

خطاف للوصول إلى الترجمات في مكونات العميل.

```tsx
const { t, locale, setLocale } = useTranslation("namespace");
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `namespace` | `string` | نعم | مساحة اسم الترجمة للاستخدام |

### Returns

| Property | Type | Description |
|----------|------|-------------|
| `t` | `(key: string, vars?: object) => string` | دالة الترجمة |
| `locale` | `string` | اللغة الحالية |
| `setLocale` | `(locale: string) => void` | دالة لتغيير اللغة |

### Example

```tsx
const { t, locale, setLocale } = useTranslation("common");

// الاستخدام الأساسي
t("greeting");  // "مرحبا"

// مع المتغيرات
t("greeting_name", { name: "جون" });  // "مرحبا، جون!"

// المفاتيح المتداخلة
t("nav.home");  // "الرئيسية"

// تغيير اللغة
setLocale("fr");
```

## getTranslation

دالة غير متزامنة للوصول إلى الترجمات في مكونات الخادم (Next.js).

```tsx
const { t } = await getTranslation("namespace");
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `namespace` | `string` | نعم | مساحة اسم الترجمة للاستخدام |

### Returns

| Property | Type | Description |
|----------|------|-------------|
| `t` | `(key: string, vars?: object) => string` | دالة الترجمة |
| `locale` | `string` | اللغة الحالية |

### Example

```tsx
// مكون الخادم
export default async function Page() {
  const { t } = await getTranslation("common");

  return <h1>{t("title")}</h1>;
}
```

## Link Component

مكون رابط يدعم اللغة لـ Next.js.

```tsx
import { Link } from "@onruntime/translations/next";

<Link href="/about">حول</Link>
```

### Props

جميع الخصائص من مكون `Link` في Next.js، بالإضافة إلى:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `locale` | `string` | اللغة الحالية | تجاوز لغة الرابط |

### Example

```tsx
// يستخدم اللغة الحالية
<Link href="/about">حول</Link>

// فرض لغة معينة
<Link href="/about" locale="fr">À propos</Link>
```

## Translation Function (t)

دالة `t` تُستخدم لاسترجاع السلاسل المترجمة.

### Signature

```tsx
t(key: string, variables?: Record<string, string | number>): string
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `key` | `string` | مفتاح الترجمة (يدعم ترميز النقاط) |
| `variables` | `object` | المتغيرات للتداخل |

### Variable Interpolation

```json
{
  "greeting": "مرحبا، {name}!",
  "items": "لديك {count} عناصر"
}
```

```tsx
t("greeting", { name: "أليس" });  // "مرحبا، أليس!"
t("items", { count: 5 });          // "لديك 5 عناصر"
```

### Nested Keys

```json
{
  "user": {
    "profile": {
      "title": "الملف الشخصي"
    }
  }
}
```

```tsx
t("user.profile.title");  // "الملف الشخصي"
```

## Type Definitions

### TranslationNamespaces

قم بتوسيع هذه الواجهة لتكون آمنة من حيث النوع للترجمات:

```tsx
declare module "@onruntime/translations" {
  interface TranslationNamespaces {
    common: typeof import("./locales/ar/common.json");
    auth: typeof import("./locales/ar/auth.json");
  }
}
```

<Callout type="info" title="فوائد TypeScript">
  مع التعريفات الصحيحة للنوع، ستحصل على إكمال تلقائي لمفاتيح الترجمة وأخطاء في وقت التجميع لمفاتيح غير صالحة.
</Callout>
