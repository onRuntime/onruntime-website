## O que é @onruntime/translations?

`@onruntime/translations` é uma biblioteca de internacionalização (i18n) leve e segura em termos de tipo, projetada para aplicativos React e Next.js. Ela oferece uma API simples para gerenciar traduções com suporte total ao TypeScript.

<Callout type="info" title="Principais Recursos">
  - Traduções seguras em termos de tipo com TypeScript
  - Suporte para React e Next.js (App Router)
  - Organização baseada em namespaces
  - Interpolação de variáveis
  - Leve e com bom desempenho
</Callout>

## Por Que Usar?

### Segurança de Tipo

Receba erros em tempo de compilação ao usar chaves de tradução inválidas:

```typescript
const { t } = useTranslation("common");

t("greeting");        // ✅ Chave válida
t("invalid_key");     // ❌ Erro de TypeScript
```

### API Simples

Apenas dois hooks para todas as suas necessidades de tradução:

```typescript
// Componentes de Cliente
const { t } = useTranslation("namespace");

// Componentes de Servidor (Next.js)
const { t } = await getTranslation("namespace");
```

### Traduções Organizadas

Mantenha suas traduções organizadas com namespaces:

## Exemplo Rápido

```tsx
// locales/en/common.json
{
  "greeting": "Olá, {name}!"
}

// Componente
import { useTranslation } from "@onruntime/translations/react";

function Welcome({ name }) {
  const { t } = useTranslation("common");

  return <h1>{t("greeting", { name })}</h1>;
}
```
