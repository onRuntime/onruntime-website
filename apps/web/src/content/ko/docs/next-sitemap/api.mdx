## generateSitemap

당신의 Next.js 애플리케이션에 대한 사이트맵을 생성해요.

```tsx
import { generateSitemap } from "@onruntime/next-sitemap/app";

const sitemap = await generateSitemap(options);
```

### Options

| Option | Type | Required | Description |
|--------|------|----------|-------------|
| `baseUrl` | `string` | 네 | 당신 사이트의 기본 URL |
| `additionalPaths` | `SitemapEntry[]` | 아니오 | 포함할 추가 URL |
| `exclude` | `string[]` | 아니오 | 제외할 glob 패턴 |
| `defaults` | `SitemapDefaults` | 아니오 | 모든 항목에 대한 기본 값 |
| `locales` | `string[]` | 아니오 | 지원하는 로케일 |
| `defaultLocale` | `string` | 아니오 | 기본 로케일 |

### SitemapEntry

| Property | Type | Description |
|----------|------|-------------|
| `url` | `string` | 페이지 URL (상대적 또는 절대적) |
| `lastModified` | `Date \| string` | 마지막 수정 날짜 |
| `changeFrequency` | `ChangeFrequency` | 페이지 변경 빈도 |
| `priority` | `number` | 우선 순위 (0.0 - 1.0) |

### ChangeFrequency

```tsx
type ChangeFrequency =
  | "always"
  | "hourly"
  | "daily"
  | "weekly"
  | "monthly"
  | "yearly"
  | "never";
```

### Example

```tsx
import { generateSitemap } from "@onruntime/next-sitemap/app";

export default async function sitemap() {
  const blogPosts = await fetchPosts();

  return generateSitemap({
    baseUrl: "https://example.com",
    defaults: {
      changeFrequency: "weekly",
      priority: 0.7,
    },
    additionalPaths: blogPosts.map((post) => ({
      url: `/blog/${post.slug}`,
      lastModified: post.updatedAt,
      priority: 0.8,
    })),
    exclude: ["/admin/*", "/api/*"],
  });
}
```

## generateRobots

robots.txt 파일을 생성해요.

```tsx
import { generateRobots } from "@onruntime/next-sitemap/app";

const robots = generateRobots(options);
```

### Options

| Option | Type | Required | Description |
|--------|------|----------|-------------|
| `baseUrl` | `string` | 네 | 당신 사이트의 기본 URL |
| `rules` | `RobotsRule[]` | 아니오 | 사용자 정의 로봇 규칙 |
| `sitemap` | `string \| boolean` | 아니오 | 사이트맵 URL 또는 제외할 경우 `false` |

### RobotsRule

| Property | Type | Description |
|----------|------|-------------|
| `userAgent` | `string` | 타겟 사용자 에이전트 |
| `allow` | `string \| string[]` | 허용된 경로 |
| `disallow` | `string \| string[]` | 불허 경로 |
| `crawlDelay` | `number` | 초 단위의 크롤링 지연 |

### Example

```tsx
import { generateRobots } from "@onruntime/next-sitemap/app";

export default function robots() {
  return generateRobots({
    baseUrl: "https://example.com",
    rules: [
      {
        userAgent: "*",
        allow: "/",
        disallow: ["/admin/", "/private/"],
      },
      {
        userAgent: "Googlebot",
        allow: "/",
        crawlDelay: 1,
      },
    ],
  });
}
```

## generateSitemapIndex

대규모 사이트를 위한 사이트맵 인덱스를 생성해요.

```tsx
import { generateSitemapIndex } from "@onruntime/next-sitemap/app";

const index = await generateSitemapIndex(options);
```

### Options

| Option | Type | Required | Description |
|--------|------|----------|-------------|
| `baseUrl` | `string` | 네 | 당신 사이트의 기본 URL |
| `sitemapsPerFile` | `number` | 아니오 | 사이트맵당 URL 수 (기본값: 5000) |

### Example

```tsx
import { generateSitemapIndex } from "@onruntime/next-sitemap/app";

export default async function sitemap() {
  return generateSitemapIndex({
    baseUrl: "https://example.com",
    sitemapsPerFile: 10000,
  });
}
```

## Type Exports

이 패키지는 편의를 위해 TypeScript 타입을 내보내요:

```tsx
import type {
  SitemapEntry,
  SitemapOptions,
  RobotsOptions,
  RobotsRule,
  ChangeFrequency,
} from "@onruntime/next-sitemap";
```
