## @onruntime/translations란 무엇인가요?

`@onruntime/translations`는 React와 Next.js 애플리케이션을 위해 설계된 경량의 타입 안전 국제화(i18n) 라이브러리입니다. 전체 TypeScript 지원을 갖춘 번역 관리용 간단한 API를 제공합니다.

<Callout type="info" title="주요 기능">
  - TypeScript로 타입 안전한 번역
  - React와 Next.js(App Router) 지원
  - 네임스페이스 기반 조직
  - 변수 보간
  - 경량 및 성능 최적화
</Callout>

## 왜 사용해야 하나요?

### 타입 안전성

유효하지 않은 번역 키를 사용할 때 컴파일 오류를 받습니다:

```typescript
const { t } = useTranslation("common");

t("greeting");        // ✅ 유효한 키
t("invalid_key");     // ❌ TypeScript 오류
```

### 간단한 API

모든 번역 요구를 위한 두 개의 훅만 있습니다:

```typescript
// 클라이언트 컴포넌트
const { t } = useTranslation("namespace");

// 서버 컴포넌트 (Next.js)
const { t } = await getTranslation("namespace");
```

### 조직된 번역

네임스페이스로 번역을 조직적으로 유지하세요:

<Files>
  <Folder name="locales" defaultOpen>
    <Folder name="en" defaultOpen>
      <File name="common.json" />
      <File name="auth.json" />
      <File name="dashboard.json" />
    </Folder>
    <Folder name="fr">
      <File name="common.json" />
      <File name="auth.json" />
      <File name="dashboard.json" />
    </Folder>
  </Folder>
</Files>

## 간단한 예제

```tsx
// locales/en/common.json
{
  "greeting": "Hello, {name}!"
}

// 컴포넌트
import { useTranslation } from "@onruntime/translations/react";

function Welcome({ name }) {
  const { t } = useTranslation("common");

  return <h1>{t("greeting", { name })}</h1>;
}
```
