## 기본 사용법

### useTranslation 훅

`useTranslation` 훅은 React 컴포넌트에서 번역에 접근하는 주요 방법이에요:

```tsx
import { useTranslation } from "@onruntime/translations/react";

function MyComponent() {
  const { t, locale } = useTranslation("common");

  return (
    <div>
      <p>현재 로케일: {locale}</p>
      <h1>{t("title")}</h1>
    </div>
  );
}
```

### 변수 보간

번역에 변수를 전달하세요:

```json
// locales/en/common.json
{
  "greeting": "안녕, {name}!",
  "items_count": "당신은 {count}개의 항목이 있습니다"
}
```

```tsx
function Greeting({ name, itemCount }) {
  const { t } = useTranslation("common");

  return (
    <>
      <p>{t("greeting", { name })}</p>
      <p>{t("items_count", { count: itemCount })}</p>
    </>
  );
}
```

## 여러 네임스페이스

같은 컴포넌트에서 여러 네임스페이스를 사용할 수 있어요:

```tsx
function Dashboard() {
  const { t: tCommon } = useTranslation("common");
  const { t: tDashboard } = useTranslation("dashboard");

  return (
    <div>
      <h1>{tDashboard("title")}</h1>
      <button>{tCommon("buttons.save")}</button>
    </div>
  );
}
```

## 로케일 변경

`setLocale` 함수를 사용해 현재 로케일을 변경하세요:

```tsx
import { useTranslation } from "@onruntime/translations/react";

function LanguageSwitcher() {
  const { locale, setLocale } = useTranslation("common");

  return (
    <select value={locale} onChange={(e) => setLocale(e.target.value)}>
      <option value="en">영어</option>
      <option value="fr">프랑스어</option>
      <option value="de">독일어</option>
    </select>
  );
}
```

## 중첩 키

점 표기법을 사용해 중첩된 번역 키에 접근하세요:

```json
// locales/en/common.json
{
  "nav": {
    "home": "홈",
    "about": "소개",
    "contact": "연락처"
  }
}
```

```tsx
function Navigation() {
  const { t } = useTranslation("common");

  return (
    <nav>
      <a href="/">{t("nav.home")}</a>
      <a href="/about">{t("nav.about")}</a>
      <a href="/contact">{t("nav.contact")}</a>
    </nav>
  );
}
```

### 키 일관성 유지

모든 로케일에서 일관된 네이밍 규칙을 사용하세요:

```json
// ✅ 좋음 - 일관된 구조
{
  "page_title": "대시보드",
  "actions": {
    "save": "저장",
    "cancel": "취소"
  }
}

// ❌ 피하기 - 일관성 없음
{
  "pageTitle": "대시보드",
  "save_button": "저장",
  "cancelBtn": "취소"
}
```
