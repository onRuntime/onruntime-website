## Kurulum

Paketi tercih ettiğiniz paket yöneticisi ile yükleyin:

```bash
# npm
npm install @onruntime/translations

# yarn
yarn add @onruntime/translations

# pnpm
pnpm add @onruntime/translations
```

## Proje Ayarı

### 1. Çeviri Dosyaları Oluşturun

Projede her dil için çeviri dosyalarının bulunduğu bir `locales` dizini oluşturun:

<Files>
  <Folder name="src" defaultOpen>
    <Folder name="locales" defaultOpen>
      <Folder name="en" defaultOpen>
        <File name="common.json" />
      </Folder>
      <Folder name="fr">
        <File name="common.json" />
      </Folder>
    </Folder>
  </Folder>
</Files>

Örnek `locales/en/common.json`:

```json
{
  "title": "Hoşgeldin",
  "greeting": "Merhaba, {name}!",
  "items": {
    "one": "{count} öğe",
    "other": "{count} öğe"
  }
}
```

### 2. TypeScript'i Yapılandırma (İsteğe Bağlı)

Tip güvenli çeviriler için bir tip tanım dosyası oluşturun:

```typescript
// types/translations.d.ts
import "onruntime/translations";

declare module "@onruntime/translations" {
  interface TranslationNamespaces {
    common: typeof import("../locales/en/common.json");
    auth: typeof import("../locales/en/auth.json");
  }
}
```

### 3. Sağlayıcıyı Ayarlayın

Uygulamanızı `TranslationsProvider` ile sarın:

```tsx
// React için
import { TranslationsProvider } from "@onruntime/translations/react";
import en from "./locales/en";
import fr from "./locales/fr";

const locales = { en, fr };

function App() {
  return (
    <TranslationsProvider locale="en" locales={locales}>
      <YourApp />
    </TranslationsProvider>
  );
}
```

## Kurulumu Doğrulama

Kurulumunuzu basit bir bileşen ile test edin:

```tsx
import { useTranslation } from "@onruntime/translations/react";

function TestComponent() {
  const { t } = useTranslation("common");

  return (
    <div>
      <h1>{t("title")}</h1>
      <p>{t("greeting", { name: "Dünya" })}</p>
    </div>
  );
}
```

Çevirilerinizi render ediliyorsa, her şey yolunda!
