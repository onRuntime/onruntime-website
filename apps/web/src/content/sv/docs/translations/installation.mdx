## Installation

Installera paketet med din föredragna pakethanterare:

```bash
# npm
npm install @onruntime/translations

# yarn
yarn add @onruntime/translations

# pnpm
pnpm add @onruntime/translations
```

## Projektinställning

### 1. Skapa översättningsfiler

Skapa en `locales` mapp i ditt projekt med översättningsfiler för varje språk:

<Files>
  <Folder name="src" defaultOpen>
    <Folder name="locales" defaultOpen>
      <Folder name="en" defaultOpen>
        <File name="common.json" />
      </Folder>
      <Folder name="fr">
        <File name="common.json" />
      </Folder>
    </Folder>
  </Folder>
</Files>

Exempel på `locales/en/common.json`:

```json
{
  "title": "Välkommen",
  "greeting": "Hej, {name}!",
  "items": {
    "one": "{count} objekt",
    "other": "{count} objekt"
  }
}
```

### 2. Konfigurera TypeScript (Valfritt)

För typ-säker översättning, skapa en typdefinitionsfil:

```typescript
// types/translations.d.ts
import "onruntime/translations";

declare module "@onruntime/translations" {
  interface TranslationNamespaces {
    common: typeof import("../locales/en/common.json");
    auth: typeof import("../locales/en/auth.json");
  }
}
```

### 3. Ställ in Provider

Lägg din app i en `TranslationsProvider`:

```tsx
// För React
import { TranslationsProvider } from "@onruntime/translations/react";
import en from "./locales/en";
import fr from "./locales/fr";

const locales = { en, fr };

function App() {
  return (
    <TranslationsProvider locale="en" locales={locales}>
      <YourApp />
    </TranslationsProvider>
  );
}
```

<Callout type="info" title="Next.js App Router">
  För Next.js med App Router, kolla in [Next.js Usage](/docs/translations/nextjs) guiden för rekommenderad installation.
</Callout>

## Verifiera installation

Testa din inställning med en enkel komponent:

```tsx
import { useTranslation } from "@onruntime/translations/react";

function TestComponent() {
  const { t } = useTranslation("common");

  return (
    <div>
      <h1>{t("title")}</h1>
      <p>{t("greeting", { name: "Världen" })}</p>
    </div>
  );
}
```

Om du ser dina översättningar återgivna, då är allt klart!
